Considerations before applying

While we strive to make room for all qualified applicants, space at MESA schools is often limited. Before you apply, consider the following.
‍
‍Availability
‍Participants should be present for the entirety of the winter school, from the beginning of activities on Monday the 17th June to Friday the 21st of June 2024.
‍
‍Funding
‍Participants are expected to pay for their own travel and accommodation. Morning tea, lunch, and afternoon tea will be covered by the school. There is no registration fee.
‍
‍Preparedness
‍Participants are expected to: 

  • Have MESA installed on a laptop or have remote access to a machine that does. Checkout the MESA installation guidelines: https://docs.mesastar.org/en/latest/installation.html
  • Be able to edit source files
  • Have a basic familiarity with the command line
  • Know how to run basic MESA simulations using inlists.

Additionally, participants will be making extensive use of Fortran to build upon existing functionalities and include new physics in MESA. Participants need not be Fortran experts, but they should know some basic coding techniques as they apply to any modern language such as control flow and loops.

In other words, as a minimum participants are expected to be familiar with how to run MESA. Additionally, we recommend that the participants familiarise themselves with the MESA output and how to build MESA inlists prior to attending the school for the best experience. For more information see:https://docs.mesastar.org/en/latest/using_mesa

Permission
‍For undergraduate (BSc), honours (MSc), and graduate (PhD) students, we will reach out to your supervisor to confirm that you meet the availability, funding, and preparedness requirements. Make sure you have your supervisor’s permission before applying.
